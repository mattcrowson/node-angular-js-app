## Setup

1. Install relevant stable version of Node and NPM for your OS.

2. Install Gulp

    & npm install -g gulp

3. Clone the repository

    $ git clone https://github.com/mattcrowson/node-angular-js-app.git

4. Download dependencies

    $ cd node-angular-js-app
    $ npm install

5. Run gulp

    $ gulp

6. Run the server

    $ node app.js
